# 支气管哮喘临床指南

## 概述
支气管哮喘是一种慢性气道炎症性疾病，表现为反复发作的喘息、气促、胸闷或咳嗽，常在夜间和清晨加重。

## 诊断标准
- 典型症状：反复喘息、气促、胸闷或咳嗽
- 可逆性气流受限：支气管扩张试验阳性（FEV1 提高 ≥12% 且 ≥200ml）
- 变应性史：过敏性鼻炎、湿疹或家族史

## 治疗建议
- 急性发作：短效β2受体激动剂（SABA）吸入
- 长期控制：吸入糖皮质激素（ICS）± 长效β2受体激动剂（LABA）
- 合并过敏：考虑抗IgE治疗

## 检查与随访
- 肺功能检测（支气管舒张试验、呼出气一氧化氮）
- 诱发因素评估（尘螨、花粉、烟雾）
- 患者教育与吸入装置规范使用
